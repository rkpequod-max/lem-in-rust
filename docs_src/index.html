📖 À propos du projet

Lem-in est un projet algorithmique qui consiste à faire passer un nombre donné de fourmis d'un point de départ à un point d'arrivée dans le minimum de tours, en évitant les collisions.


Cette version Rust offre :

  • ✅ Sécurité mémoire garantie (pas de segfault, pas de buffer overflow)
  • ✅ Performance optimale grâce au compilateur Rust
  • ✅ Compilation WASM native avec wasm-pack
  • ✅ Code moderne et maintenable avec le système de types Rust

🏗️ Structures de données

Room (Salles)

Représente une salle avec son nom, ses coordonnées, son statut (start/end) et ses connexions.

struct Room {
    name: String,
    coords: (i32, i32),
    is_start: bool,
    is_end: bool,
    links: Vec<String>,
}

Farm (Ferme)

Conteneur principal avec le nombre de fourmis et la liste des salles.

struct Farm {
    num_ants: usize,
    rooms: HashMap<String, Room>,
}

BFS & Pathfinding

Algorithme de Breadth-First Search optimisé pour trouver les chemins les plus courts.

fn find_shortest_path(
    farm: &Farm,
    start: &str,
    end: &str
) -> Option<Vec<String>>

⚙️ Algorithme

1. Parsing de la carte

Lecture et validation de l'input : nombre de fourmis, salles (nom x y), et liens (room1-room2).

2. Recherche de chemins (BFS)

Parcours en largeur pour trouver tous les chemins possibles du start à l'end, en évitant les collisions.

3. Optimisation du flux

Calcul du nombre optimal de fourmis par chemin pour minimiser le nombre total de tours.

4. Simulation

Exécution tour par tour du déplacement des fourmis avec affichage des mouvements.

🔧 Compilation & Déploiement

Le projet utilise GitHub Actions pour compiler automatiquement le code Rust en WebAssembly et déployer le site.

Workflow automatique :

  1. Push sur la branche main
  2. Installation de Rust et wasm-pack
  3. Compilation WASM : wasm-pack build --target web --release
  4. Déploiement sur GitHub Pages

Commandes locales (optionnel) :

# Installer wasm-pack
curl https://rustwasm.github.io/wasm-pack/installer/init.sh -sSf | sh

# Compiler en WASM
wasm-pack build --target web --release

# Tester localement
python3 -m http.server 8000
